# SAEP
Repositório criado para fins de teste e preparação para o SAEP

Requisitos Funcionais (RF)

RF001 – Autenticação de Usuários
O sistema deve permitir que usuários realizem login utilizando nome de usuário e senha.

RF002 – Validação de Credenciais
O sistema deve validar login e exibir mensagem caso usuário ou senha estejam incorretos.

RF003 – Logout
O sistema deve permitir que o usuário encerre a sessão e seja redirecionado para a tela de login.

RF004 – Cadastro de Produtos 
O sistema deve permitir:
Cadastrar novo Produto


Editar material existente


Excluir Produtos


Pesquisar Produtos
Campos obrigatórios:
Nome do Produto
Quantidade
Estoque mínimo

RF005 – Listagem e Busca de Materiais
O sistema deve apresentar a lista de materiais cadastrados ao abrir a tela e permitir pesquisar por nome.

RF006 – Ordenação Alfabética
A lista de materiais deve ser exibida em ordem alfabética automaticamente.


RF007 – Tela Principal
Após login deve exibir:
Nome do usuário logado


Botão de logout


Atalhos para:


Cadastro de Produtos


Gestão de Estoque
